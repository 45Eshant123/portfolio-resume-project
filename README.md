## Overview

This is a modern, interactive portfolio website designed to showcase the skills, projects, services, and achievements of Eshant Verma, a Web Developer & Programmer. The website serves as a personal brand platform, helping recruiters and clients quickly understand technical capabilities, past work, and ways to get in touch.

## Key Features

- **Hero Section**: Introduction with name, tagline, and call-to-action buttons.
- **Skills Section**: Highlights technical skills including C, C++, Java, HTML, CSS, JavaScript, and Creative Design.
- **Projects Section**: Showcases top projects with titles, descriptions, tools used, GitHub links, and live demos.
- **About Section**: Brief introduction about the developer, journey, goals, and passion for tech.
- **Services Section**: Lists services such as Web Development, UI/UX Creative Design, Frontend Development, and Landing Page Creation.
- **Certifications Section**: Displays completed courses and certifications.
- **Contact Section**: Simple form and social links for communication.
- **3D Boy / Animated Robot**: Planned feature with a 3D animated character that follows scroll.
